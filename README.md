Data Professional Survey Breakdown — Power BI Project

#Overview
This project analyzes survey data from data professionals to uncover trends related to job satisfaction, salary, career paths, tools used, and skill growth.
The project consists of two parts:
1)Raw Dataset – Provided in Excel format (Power BI.xlsx)
2)Power BI Dashboard – Interactive analytical report (Final_Project.pbix)
This repository demonstrates skills in data cleaning, ETL, modeling, visualization, and dashboard development using Power BI.

1. Power BI.xlsx
This file contains the raw, uncleaned dataset used for the project.
The dataset includes fields such as:
Job title
Country
Salary
Favorite programming languages
Tools used
Level of job satisfaction
Experience levels
Industry and work environment
This file represents the data preparation starting point.

2. Final_Project.pbix
This is the final Power BI dashboard, built after:
Cleaning and transforming the raw dataset
Creating calculated columns & measures
Building data models
Designing visual insights

#Dashboard Highlights
The Power BI report includes interactive visuals covering:
A)Demographics
Gender distribution,
Country and region insights,
Job titles and roles.

B)Career Information
Experience levels,
Career transitions,
Work-life balance metrics

C)Salary Analysis
Salary distribution,
Salary comparison by role,
Salary vs. experience

#Survey KPIs
Average salary,
Overall satisfaction score,
Most popular tools & languages,
The dashboard uses slicers and interactive filters to let users explore the data dynamically.
